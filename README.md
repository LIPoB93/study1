# 요한계시록 성구 암기 오프라인 웹앱

## 사용 방법
1. 이 폴더 전체를 GitHub 저장소 또는 Vercel에 올립니다.
2. 배포된 주소를 iPhone 또는 iPad Safari로 엽니다.
3. 공유 버튼 → `홈 화면에 추가`를 누릅니다.
4. 최초 1회 접속 이후에는 오프라인에서도 실행됩니다.

## 포함 범위
- 요한계시록 7장 전체
- 요한계시록 14장 전체: 빈칸 채우기 기본
- 요한계시록 15장 전체: 빈칸 채우기 기본
- 요한계시록 1:1~3, 10:10~11, 20:4~6, 22:1~5, 22:18~19

## 저장 방식
- 진도와 오답은 브라우저 localStorage에 저장됩니다.
- `기록 관리` 화면에서 백업 파일을 내려받거나 불러올 수 있습니다.
- 브라우저 데이터 또는 앱 데이터를 삭제하면 기록이 사라질 수 있습니다.
