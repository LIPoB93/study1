# 성구 암기 훈련 v5

개역한글 성경 66권 전체를 포함한 오프라인 PWA입니다.

## 주요 기능
- 성경/장/시작 절/끝 절을 선택해 학습 범위 저장
- 여러 구간을 하나의 시험으로 묶기
- 토씨 직접 입력 검사 및 틀린 글자 표시
- 오답노트 누적, 오답만 재시험
- 토씨 시험 통과 달력 기록
- 모든 기록은 각 기기의 localStorage에 저장

## 배포
압축을 풀고 파일 전체를 GitHub 저장소 최상단에 올린 뒤 Vercel로 배포하세요. 최초 접속 후 서비스워커가 전체 본문을 캐시하므로 오프라인에서도 작동합니다.

본문: 성경전서 개역한글판. 본문은 임의 수정 없이 사용합니다.
